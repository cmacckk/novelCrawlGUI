# 爬取小说(界面)

## 安装

### 可执行文件

从[release](https://github.com/cmacckk/novelCrawlGUI/releases)下载对应系统版本文件，直接运行即可

### 源码运行

```bash
python3 -m pip install -r requirements.txt
python3 main.py
```
